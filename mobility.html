Mobility Print

Mobility Print is currently in beta




Mobility print is a new way to connect to RCS printers at RPI. It is primarily for mobile devices and Mac OS computers but Windows users can use it as well.


  • Android
  • iOS
  • Chrome OS
  • macOS
  • Windows

Installation instructions can be found here (Still in Beta link is not active yet). When prompted for a password to add the printer, use your RCS username and password.

  • Device must be connected to the RPI network, RPI wireless or via VPN for printing to work
  • Do not use mobility print on windows or mac computers that have more than 1 user. It is likely that prints will be charged to the user that installed the printer.
  • Chrome OS does not encrypt the print jobs in transit
  • Plotting from mobile devices is not currently recommended
